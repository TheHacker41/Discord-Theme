# Discord Themes

This Repository Contains Custom Discord Themes Made For **Vencord** Using BasicBackground By **DevilBro**.

---

## Requirements

You Must Have **Vencord** Installed To Use These Themes.

If You Do Not Have It Installed, Download It Here:  
https://vencord.dev/download

---

## Installation Guide

1. Open **Discord**
2. Go To **Vencord Settings**
3. Open The **QuickCSS Editor**
4. Paste ONE Of The Theme Imports Below
5. Save The Changes
6. Done

---

## Available Themes

@import url('https://raw.githubusercontent.com/TheHacker41/Discord-Theme/refs/heads/main/red.css');

---

## Optional: Improved User Panel

If You Want The Improved User Panel CSS, Make Sure This Import Is Placed **At The Top** Of Your QuickCSS:

```css
@import url('https://raw.githubusercontent.com/TheHacker41/Discord-Theme/refs/heads/main/userpanel.css');
```

---

## Important Notes

- These Themes Are Designed Specifically For **Vencord**
- Only Use **One Main Theme Import** At A Time
- The User Panel CSS Can Be Combined With Any Theme
- Make Sure Imports Stay At The Top Of Your QuickCSS
